# Demon's Destiny

A 2D beat-em-up RPG following Vraak, an Asura descending through the layers of Naraka to uncover the truth behind his father's death.

## About

**Demon's Destiny** is a 2D side-scrolling beat-em-up RPG set in a world drawn from Indian mythology. Players take control of Vraak, an Asura whose journey begins with his father's death and pulls him through the descending layers of Naraka in search of the truth — only to uncover something far more shocking than he expected. Along the way, Vraak crosses paths with a range of mythological beings — Daityas, Rakshasas, Vanaras, Nagas, and more — each shaping the world and story around him.

## Current State

This is a vertical slice / prototype. Core combat and movement systems are implemented, with one enemy archetype currently in place to demonstrate the combat framework. Additional enemies, levels, and story content are in active development.

## Tech Stack

- **Engine:** Unity
- **Language:** C#
